Проблемы при формировании документов

Форма отображает список учётных документов, для которых не удалось сформировать электронные документы.

В колонке «Результат» выводится гиперссылка с кратким описанием ошибок. При переходе по гиперссылке открывается помощник по устранению ошибок. После исправления ошибок в колонке появляется гиперссылка «Сформирован», по которой можно открыть форму просмотра сформированного электронного документа.

Если электронный документ был сформирован по команде «Отправить электронный документ», то на командной панели формы отображается кнопка «Подписать и отправить документы». Она позволяет повторно сформировать, подписать и отправить электронные документы.
